Anleitung

Hier finden Sie die relevantesten Informationen, die Ihnen die Bedienung der Gartenhilfe bestmöglich erleichtern.

Sensoren

In diesem Reiter bekommen Sie die Möglichkeit, ausgelesene Daten Ihrer Sensoren anzeigen lassen zu können. Dementsprechend werden auch die passenden Einheiten dazu angezeigt.

Wetter

Mit Hilfe unserer implementierten Wetter-API ist es Ihnen jederzeit möglich, aktuellste und genaueste Wetterinformationen Ihres Standorts abzurufen. Sie können ebenso das prognostizierte Wetter der kommenden Tage so ganz komfortabel abrufen.

Tipps

Hier finden Sie eine riesige Auswahl an verschiedenen Pflanzenarten und Beispielbildern zu diesen Pflanzen. Nachdem Sie eine Pflanze ausgewählt haben, werden Ihnen die wichtigsten Ansprüche der jeweiligen Pflanze angezeigt.
